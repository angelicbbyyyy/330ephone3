✨ Version 2.20 Update

To ensure better service and protect creators, please read the following changes carefully.

  • 🔐 Login System:
    Added a lightweight login system to prevent unauthorized reselling and protect the community. Get your credentials in the EPhone Discord using /小手机.
    * This process is only used for eligibility verification. No personal private data is collected.
  • 🔔 System Notifications:
    Background browser notifications are now supported. Open API Settings and grant notification permission.
    Enable system notifications in API Settings to receive alerts in the background.
  • 🔌 Local API Fetching:
    Added beta support for fetching local API models. If automatic fetching fails, manual entry still works. Important: after fetching, you must select the model from the dropdown. The model name will no longer be auto-filled into the input.
    If model fetching fails, manual input still works.
  • 📚 Green River & Mail:
    Includes a novel-reading experience plus an email system that can move the story forward. Sharing emails with characters can trigger reactions.
    Green River supports reading flow, and Mail can be used for story progression.
  • 🛠️ Other Improvements:
    Added support for more sticker import formats, improved CSS beautification for document imports, added Narrator mode (/n or /旁白), and added a Todo list for private chats only.
    Better sticker support, improved document import styling, narrator mode, and a private-chat todo list.
  • 🛡️ About the PIN Code:
    The PIN check is entirely local logic and does not steal privacy. See the notes in Tail Town or the beginner tutorial to learn how to get it. Each browser/device has a unique PIN, so revealing the device ID is not a security risk.
    The PIN is local, device-specific, and safe.
  • 🚫 Do Not Share Decrypted Content:
    Do not share decrypted character-card content or Bunny K keys. Bunny K and 330 use different decryption URLs, so follow the instructions inside the card carefully.
    Do not repost decrypted content or keys.
  • 🔒 Tavern Import Lock:
    A lock was added for importing Tavern cards, presets, and world books to protect creators. This does not affect other features. If you need to unlock it, check the designated section in Tail Town for instructions.
    Import protection exists to protect authors and preset creators.
