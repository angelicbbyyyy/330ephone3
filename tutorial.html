Welcome to the EPhone Tutorial

This guide covers the most commonly used EPhone features and setup steps.

Feature Guide (click to expand)

1. Full-Screen Setup

If anything is covered or clipped on iOS or Android, check the layout customization section first.

1.1 iOS
  • Add the page to your Home Screen.
  • On iOS, go to Appearance settings and turn off the top bar if needed.
1.2 Android Chrome / Edge
  • First open Settings and allow these two permissions.
  • Open the page, tap the three dots in the top-right corner, choose Add to Home Screen, then select the first install/download option. It may take a moment.
  • In Appearance settings, enable the top status bar if you want to see battery information.
2. Feature Overview
2.1 Bottom Controls
  • As shown, the last two buttons are rewind and fast-forward.
2.2 Top Controls
  • 2.2.1 The button next to Listen Together opens Long-Term Memory.
  • 2.2.2 Long-Term Memory

    • Open Long-Term Memory to manage saved memory. If auto-summary fails, use Summarize manually. Use the plus button to add your own entries. If memory gets too large, use Refine and choose the target length. Refining requires confirmation before it overwrites the original memory.

2.3 Long-Press Actions
2.5 Regex / Renderer
  • Open the Renderer from the home screen and group your regex rules by category. Some need to be separated into different entries.

    • For the Xiaohongshu-style world book, formatting tends to drift a lot. To avoid wasting tokens, send your template and the malformed content to Gemini / DS / GPT and ask it to reformat strictly to the template without changing any content.

2.6 Video Calls
  • After a video call ends, the app will try to generate long-term memory automatically. If it fails, open Messages and retry from there.
2.7 Listen Together
  • Song links found by search may only stay valid for around 12 hours. Use Clean Up to remove expired links.
2.8 Background Actions
  • Use background actions and independent-action cooldowns wisely. Characters can greet you on their own. If you have many characters, use a longer interval; if you have fewer, shorten it. Independent cooldown controls the gap between the first and second autonomous actions.
2.9 Thoughts
  • In a private chat, tap the avatar to open it. In the record list, tap a card lightly to reveal the delete button.
2.10 Sticker Formats
  • User Catbox batch upload

    • Use the Catbox URL and include the code that represents each emotion, for example:
    https://files.catbox.moe/...
    Anxious: 2a9wte.jpeg

  • User non-Catbox batch upload

    • Example: heart launch effect: https://i.postimg.cc/KjGsZ2zc/image.gif

2.11 Chat Button Order
  • Drag buttons into the order you prefer.
2.12 Presets
  • You can upload JSON-format presets. These are mainly useful in offline mode.

    • I have mainly used them for writing style and anti-interruption behavior. Gemini tends to follow person-setting instructions in presets more reliably than DS.

2.13 Memory Cleanup
  • Clean Redundant Data

    • This may clear NPC avatars in group chats. That is normal.

  • Advanced Data Cleanup

    • You can target specific characters for cleanup. The same idea applies to the feed page.

2.14 Character Keeps Replying to Feed Posts
  • You need to tell the character that you already saw the comment and want to talk normally. If you posted or replied to them, the system keeps reminding them about it, so you need to overwrite that context in their memory to stop the repeated feed-comment behavior.
3. If Douban or Phone Inspection Fails to Generate

CLI generation can currently be unreliable. Usually switching to a different API or model helps. I have tested Gemini official keys, Hugging Face proxies, and DS v3.1.

4. API Notes
4.1 Google Official Direct Endpoint

    https://generativelanguage.googleapis.com/v1beta/models

4.2 Primary and Secondary APIs

    Use the secondary API wisely. A lightweight model such as Gemini 2.5 Flash is usually enough there, so do not waste stronger models on summarization work.

5. CSS Customization

Make sure to adjust the values yourself. Every phone size is different, so copy the closest snippet and fine-tune it for your device.
